description: Get Next.js project structure and configuration
---

Use the Next.js MCP server to retrieve comprehensive project information.

Call `get_project_metadata` and present:
- Next.js version
- Dev server URL and port
- Project structure overview
- Key configuration settings
- Enabled features (App Router, experimental flags, etc.)

Keep the output concise and actionable. Highlight any configuration issues or recommended optimizations.
